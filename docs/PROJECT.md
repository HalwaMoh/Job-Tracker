# Job Tracker System

## Overview

Job Tracker System is a web application that helps students and job seekers organize and track internship and job applications throughout their job search.

Instead of using spreadsheets, notes, emails, and bookmarks, users can manage everything from one dashboard.

---

## Problem

Job seekers often lose track of:

- Where they applied
- Application deadlines
- Interview schedules
- Follow-up reminders
- Application status
- Resume versions
- Notes from interviews

Managing dozens of applications becomes difficult and increases the chance of missing opportunities.

---

## Solution

The application provides a centralized platform where users can:

- Track job applications
- Update application status
- Receive reminders
- Store notes
- Monitor progress
- View analytics

---

## Target Users

- University students
- Internship seekers
- Fresh graduates
- Job seekers

---

## Goals

- Make job searching organized.
- Reduce missed deadlines.
- Help users stay motivated.
- Provide insights into job search progress.
